## CSS

1. 인라인

   `<h1 style="color: blue ; font-size: 100px;">Hello</h1>`

2. 내부참조

   `<style> h1 { color : blue; font-size: 100px; } </style>`

3. 외부참조

   외부 .css 파일을 만들어 Link를 통해 불러오기 (class 값을 통해서 스타일 불러올 수 있음)

   `<link rel="stylesheet" href=".css">`

* 지정해서 정의해줄때
  * ` div > p > span {color:red;}`
